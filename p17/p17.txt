Merge Two Sorted Lists

Input 
The first line contains an integer 𝑛, the number of elements in the first list.
The second line contains 𝑛 integers in non-decreasing order.
The third line contains an integer 𝑚, the number of elements in the second list.
The fourth line contains 𝑚 integers in non-decreasing order.

Output
